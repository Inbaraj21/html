Acess specifier

Definition of acess specifier

Java acess specifier(also kown as visibility specifier) regulate acess to class,fields and methods in java.

these specifier determine whether a field or method in class,can be used or invoked by another method in a another class or sub class.

acess specifier can be used to restrict access.

types of access specifiers

  1. public
  2. private
  3. protected
  4. default

public

  • public specifier access the highest level of accessibility.
  • classes ,methods,and fields declared as public can be acessed from any class in the java program,these class are same package or in another package.

private

  • private specifier achieves the lowest level of accessibility.
  • private methods and fields can only be accessed within the same class to which the methods fields belong.
  • using private specifiers we can achieve encapsulation and hide data from the outside world.

protected

  • methods and fields declared as protected can only be accessed by the subclasses in the other packages or any class within the package of the protected member class.
  • the protected acess specifier cannot be applied to class and inheritances.

default

  • using default specifier we can acess class ,method ,or field which belongs to same packages,but not from outside this package.
